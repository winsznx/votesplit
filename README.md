# VoteSplit

Quadratic voting system with democratic fund allocation on Base and Stacks blockchains.

## Features

- Create funding proposals
- Quadratic voting mechanism
- Voice credit system
- Multiple voting options
- Deadline-based voting periods

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Base**: Solidity 0.8.20, Foundry, Reown wallet
- **Stacks**: Clarity v4, Clarinet, @stacks/connect

## Getting Started

```bash
pnpm install
pnpm dev
```
